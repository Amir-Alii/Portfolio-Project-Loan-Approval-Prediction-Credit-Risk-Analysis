# Portfolio-Project-Loan-Approval-Prediction-Credit-Risk-Analysis
AI Engineer working in the Retail Credit Risk Analytics team of a commercial bank.
The bank receives thousands of personal loan applications every month. Each application contains demographic, financial, and credit-behavior information about the applicant. The bank’s underwriting team wants to:
Understand what factors influence loan approvals
Build a predictive model that estimates the probability of loan approval
Design a decision rule (approval/decline threshold) based on business risk
Ensure the solution is explainable, fair, and production-ready
Task is to use the provided dataset (csv) to simulate a real-world bank credit underwriting project, covering the full AI lifecycle.
